The script here reads the RSS feed for the ZMan. Extraction of useful HTML is
performed by the READXMLDOM Java program.

See the "ChicagoBoyz" folder for a copy of the necessary "ReadXMLDOM.class" file.
The source for that Java program is available in the "java" folder, if
you wish to compile it yourself.

This script is depending on the presence of the "wget" and "java" commands.
